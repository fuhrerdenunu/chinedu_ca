# chinedu.ca

Personal website and photography portfolio. Built with Jekyll and hosted on GitHub Pages.

**Live site:** [chinedu.ca](https://chinedu.ca)

## Tech stack

- [Jekyll](https://jekyllrb.com/) — static site generator
- [GitHub Pages](https://pages.github.com/) — hosting and deployment
- [GitHub Actions](https://github.com/features/actions) — automated build and deploy

## Local setup

**Prerequisites:** Ruby 3.0+, Bundler

```bash
# Install dependencies
bundle install

# Run locally
bundle exec jekyll serve
```

Site will be available at `http://localhost:4000`.
